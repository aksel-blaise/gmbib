3.7 Social structure - collaboration

3.7.1 Author collaboration

Scientific collaborations are plotted where nodes are the authors and links are co-authorships.

# author collaboration network
collab1 <- biblioNetwork(df, analysis = "collaboration", network = "authors", sep = ";")
# network plot
collabnet1 = networkPlot(collab1, n = 75, Title = "Author Collaboration", type = "auto", size = 15, size.cex = T, edgesize = 5, labelsize = 0.5)

# descriptive analysis of author collaboration network
collabnetstat <- networkStat(collab1)
summary(collabnetstat, k = 15)
